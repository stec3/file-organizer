# file-organizer

A Python script that automatically organizes files into folders based on their type and extension.

## How to use
1. Clone the repo
2. Run 'python organizer.py'
3. Drag and drop the folder you want to organize

## Supported files type
- Images: jpg, jpeg, png
- Documents: pdf, docx
- Video: mp4
- Music: mp3
- Archives: zip
